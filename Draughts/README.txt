Open program as follows:
./Draughts

The menu window below will appear when program opens. Start by adding
minimum 2 players to the roster before you play game.
--------------------------------
English Draughts
----------------
1) Add Player to the system
2) Play Game
3) Exit Game
Please enter your choice: 
---------------------------------

When you have added two players to the roster and you have entered choice two
for Play Game the game window below will appear.

The checker’s location may be referred to by two digits in a row – firstly 
the column and then the row. So “1,1” would refer to the top left cell and 
“8,1” to the top right, “1,8” to the bottom left, and so on.

Furthermore can the game rules be found on the url address below.
